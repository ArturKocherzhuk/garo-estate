## 📌 Анотація (Про проєкт)
Цей репозиторій містить вихідний код та матеріали практичної частини курсового проєкту. Метою роботи є розробка сучасного, зручного та повністю адаптивного вебсайту-каталогу для агенції нерухомості "Garo Estate" з використанням актуальних технологій фронтенд-розробки.

## 🎯 Мета та завдання курсової роботи
* **Об'єкт розробки:** процес створення сучасного адаптивного вебсайту.
* **Предмет розробки:** вебсайт-каталог агенції нерухомості, реалізований за допомогою HTML, CSS та JavaScript.
* **Основне завдання:** спроєктувати структуру, розробити дизайн-макет та реалізувати клієнтську частину (Front-end) вебдодатка, що дозволяє користувачам зручно шукати, переглядати та підбирати об'єкти нерухомості (вілли, котеджі, квартири) на різних пристроях.

## 📖 Структура пояснювальної записки (Теоретична частина)
Курсова робота складається з пояснювальної записки та практичної реалізації. Короткий огляд розділів курсової:

### Розділ 1. Аналітичне обґрунтування та вибір технологій
* Сформовано основні вимоги до програмного продукту та функціоналу каталогу.
* Обґрунтовано вибір інструментів розробки (редактор **Visual Studio Code**, інструменти відлагодження **Chrome Developer Tools**, система контролю версій **Git/GitHub**).
* Описано вибір базових технологій: **HTML5** для семантичної структури, **CSS3** для оформлення та **JavaScript (ES6+)** для інтерактивності.

### Розділ 2. Проєктування та адаптивна верстка
* Описано архітектуру проєкту: вебсайт складається з головної сторінки (10 логічних секцій), сторінки каталогу та динамічної сторінки окремого об'єкта.
* В основі розробки лежить методологія **«Mobile First»** — інтерфейс спочатку створювався під мобільні пристрої, а потім адаптувався під планшети та ПК за допомогою **CSS-медіазапитів** (наприклад, зміна сітки популярних пропозицій при розширенні екрана до 768px і вище).
* Відмова від важких шрифтових іконок на користь вбудованих **SVG-зображень** для оптимізації швидкості завантаження.

### Розділ 3. Програмна реалізація логіки (JavaScript)
* **Динамічний рендеринг:** Дані про нерухомість зберігаються у вигляді масиву об'єктів у JS. Каталог будується динамічно, що дозволяє легко оновлювати асортимент.
* **Система фільтрації:** Написано власні скрипти для пошуку та сортування об'єктів за ціною, площею (за допомогою інтерактивних повзунків), статусом та наявністю додаткових зручностей.
* **Сторінка об'єкта (`property-single.html`):** Створено механізм читання GET-параметрів з URL (наприклад, `?id=3`), завдяки якому на сторінці динамічно відображаються характеристики, ціна та галерея конкретної вілли.
* **Слайдери та анімації:** * Слайдер відгуків реалізовано мовою JavaScript.
  * Динамічні лічильники у секції "Показники діяльності" реагують на подію скролінгу вікна та анімовано підраховують числа від нуля до заданого значення.
